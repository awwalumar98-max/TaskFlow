# TaskFlow — Simple Task Manager

A lightweight to-do / task manager built with plain HTML, CSS, and JavaScript. No installation required — just open `index.html` in any browser.

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
3. That's it!

## 📁 Project Structure

```
taskflow/
├── index.html      # Main HTML page
├── style.css       # All styles
├── app.js          # All JavaScript logic
├── README.md       # This file
└── CONTRIBUTING.md # How to contribute
```

## ✅ Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Edit existing tasks
- Filter by All / Active / Completed
- Task counter
- Clear completed tasks
- Saves tasks to localStorage (persists on refresh)
- Responsive design (mobile-friendly)
- Smooth animations
